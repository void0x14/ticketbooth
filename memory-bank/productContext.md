# Product Context

## Purpose
Ticket Booth allows users to build a watchlist of movies and TV shows, keep track of watched titles, and find information about the latest releases. It is a management tool, not a content consumption (streaming/downloading) platform.

## Goals
- Provide a clean, native experience for movie/show tracking.
- Maintain high performance even with large libraries.
- Ensure security and modern code standards.

## Target Audience
Users who want a professional, Linux-native application to organize their media consumption habits.

## Key Enhanced Features (by void0x14)
- **Widget Recycling**: Optimized memory usage during scrolling.
- **Modern Architecture**: Refactored codebase, transitioned to English as primary development language.
- **Stability**: Fixed standard library namespace conflicts (e.g., `logging`).
- **Security**: Parameterized SQL queries.
